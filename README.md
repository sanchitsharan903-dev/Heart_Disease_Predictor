# ❤️ Heart Disease Predictor (Based on ML)

## 🔗 Live App
https://cardio-insight-ml.streamlit.app

## 📌 Overview
This project is a machine learning-based web application that predicts the risk of heart disease using medical parameters.

## 🚀 Features
- Real-time heart disease prediction
- Risk percentage calculation
- Interactive UI with Streamlit
- Feature importance graph
- Clean and modern design

## 🧠 Model Used
- XGBoost Classifier
- Accuracy: ~88%

## ⚙️ Tech Stack
- Python
- Scikit-learn
- XGBoost
- Streamlit

## 📊 Dataset
- UCI Heart Disease Dataset
